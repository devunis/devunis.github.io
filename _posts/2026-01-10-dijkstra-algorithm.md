---

## 💻 구현

```python
def solve(n):
    # 솔루션 구현
    result = []
    for i in range(n):
        result.append(i)
    return result

# 사용
print(solve(10))
```

---

## 📊 시간 복잡도

- **최선**: O(n)
- **평균**: O(n log n)  
- **최악**: O(n²)
